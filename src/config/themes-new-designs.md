## 主题 1：赛博霓虹 (Cyberpunk Neon)

### 图标
🌆

### 配色方案
```
--accent:       #ff2d78  (霓虹品红)
--accent-alt:   #00f0ff  (电解青)
--accent-warn:  #ffd700  (信号黄)
--bg:           #0a0a0f  (极深黑紫)
--surface:      #12121a  (深紫灰)
--text:         #f0f0ff  (冷白)
--muted:        #8888aa  (紫灰)
--border:       rgba(255,45,120,0.15)
```

### 设计风格描述
受 Blade Runner / 攻壳机动队启发的赛博朋克风格。深色背景带有紫色调，霓虹品红作为主色，电解青和信号黄作为辅助色。卡片带有发光边框效果，标题使用渐变色文字。背景可叠加微弱的网格线 + 扫光动画，营造都市夜景氛围。整体视觉冲击力强、辨识度极高。

### 适合的 JS 特效
- **光标类型**: `follower` — 方形跟随点 (8px)，悬停时变成发光圆环
- **视差**: 开启 — 背景叠加两层 parallax：慢速网格层 + 快速光晕层
- **翻转动画**: 卡片 hover 时 3D 翻转 (rotateY)
- **revealStyle**: `slide-up` (带轻微的缩放)
- **额外特效**: 按钮霓虹扫光动画、标题闪烁光标（打字机风格）

### 布局模式
`light-tech`

### 额外 CSS 要点
- 卡片带有 `box-shadow` 发光效果（霓虹色羽化）
- 背景两层径向渐变模拟城市灯光
- 标题文字使用 `linear-gradient` 渐变

## 主题 2：毛玻璃 (Frosted Glass / Glassmorphism)

### 图标
🫧

### 配色方案
```
--accent:       #7c3aed  (紫罗兰)
--accent-soft:  rgba(124,58,237,0.08)
--bg:           #f5f3ff  (极淡紫白)
--surface:      rgba(255,255,255,0.35)
--surface-strong: rgba(255,255,255,0.7)
--text:         #1e1b4b  (深紫黑)
--muted:        #8b83ba  (中紫灰)
--border:       rgba(124,58,237,0.1)
```

### 设计风格描述
现代玻璃质感风格，受 Apple / Glassmorphism 趋势影响。所有卡片、导航栏使用半透明背景 + 模糊效果（backdrop-filter），叠加微妙的渐变背景（浅紫到淡蓝）。整体通透、轻盈、有层次感。阴影柔和。这种风格在简历网站中很少见，视觉上非常独特。

### 适合的 JS 特效
- **光标类型**: `trail` — 大号透明环 (56px)，带有紫色渐变
- **视差**: 关闭（避免干扰玻璃层次感）
- **翻转动画**: 卡片轻微浮动（浮动动画，非翻转）
- **revealStyle**: `fade` (渐变出现，匹配玻璃的柔和感)
- **额外特效**: 背景慢速渐变动画、卡片悬浮微动效

### 布局模式
`light-tech`

### 额外 CSS 要点
- 卡片 `background: rgba(255,255,255,0.3); backdrop-filter: blur(16px)`
- 背景使用两层大半径渐变圆
- 导航栏模糊度更高 `blur(20px)`
- 圆角较大 (16px)

## 主题 3：暖色复古 (Warm Retro)

### 图标
🕰️

### 配色方案
```
--accent:       #d97706  (琥珀橙)
--accent-soft:  rgba(217,119,6,0.08)
--bg:           #fdf4e3  (旧纸米色)
--surface:      rgba(255,255,245,0.7)
--text:         #451a03  (深棕)
--muted:        #92400e  (中棕)
--border:       rgba(217,119,6,0.12)
```

### 设计风格描述
1970s 复古风格，温暖的琥珀色系搭配旧纸张纹理背景。灵感来自中古世纪的油画画册和老式打字机排版。标题使用 serif 字体（如 Playfair Display），正文使用衬线体或无衬线体混搭。卡片带有轻微的手绘边框阴影，整体给人温暖、可信任、有故事感。

### 适合的 JS 特效
- **光标类型**: `trail` — 琥珀色暖光光环
- **视差**: 关闭
- **翻转动画**: 卡片 hover 时轻微上浮 + 阴影加深
- **revealStyle**: `slide-up` (慢速，缓出)
- **额外特效**: 页面加载时纸纹理淡入、标题字母逐个染色

### 布局模式
`classic` (经典布局更适合复古展示)

### 额外 CSS 要点
- 标题使用 `Playfair Display` serif 字体
- 背景可选叠加 subtle 的纸张噪点纹理
- 卡片边框使用虚线或双线边框增加手工感
- 按钮使用填充底色 + 深色文字

### 设计风格描述
工程图纸风格的冷色科技设计。以冰蓝为主色调，搭配极浅的蓝白背景。借鉴蓝图（blueprint）的网格线元素。整体冷静、精确、高效，传达出工程师思维和专业感。卡片设计简洁，使用细边框 + 直角或小圆角，标题使用 engineering 风格的编号系统。

### 适合的 JS 特效
- **光标类型**: `follower` — 小号 (10px) 精准圆点，匹配工程感

### 布局模式
`light-tech`

### 额外 CSS 要点
- 背景使用 `repeating-linear-gradient` 绘制极淡网格线
- 字体可使用系统无衬线 + 等宽字体混搭
- 按钮直角设计

## 主题 5：森之 Canopy (Forest Canopy)

### 图标
🌲

### 配色方案
```
--accent:       #4d7c0f  (苔藓绿)
--accent-soft:  rgba(77,124,15,0.1)

--muted:        #7a8a6e  (灰绿)
```

### 设计风格描述
沉浸式森林自然风格。深绿色背景模拟森林底层，苔藓绿作为主色。搭配木质暖棕作为辅助色。灵感来自日本屋久岛的苔藓森林。卡片使用有机的圆角，阴影柔和如树影。整体氛围宁静、深邃、自然，与科技主题形成强烈反差。

### 适合的 JS 特效
- **光标类型**: `follower` — 绿色微光圆点，悬停时如露珠扩散
- **视差**: 开启 — 背景树叶/光斑缓慢飘移
- **翻转动画**: 卡片 hover 时产生"生长"动画（从底部升起的绿色线条）
- **revealStyle**: `fade` (柔和淡入，如雾气渐散)
- **额外特效**: 背景光点粒子模拟"丁达尔效应"光束

### 布局模式
`classic` (经典布局更适合展示自然感)

### 额外 CSS 要点
- 背景可叠加树叶轮廓的 SVG 装饰
- 卡片 hover 时左侧绿色生长条动画（类似 dark-olive 但颜色不同且更柔和）
- 标题使用稍粗的字体，间距较大
- 底部可添加森林动物足迹的 SVG 装饰

## 主题 6：深海深渊 (Abyssal Depths)

### 图标
🐋

### 配色方案
```
--accent:       #06b6d4  (荧光青)
--accent-alt:   #f97316  (珊瑚橙)
--bg:           #020617  (深渊黑蓝)

--text:         #e0f2fe  (水泡白)
--muted:        #5b8ba0  (雾蓝)

```

### 设计风格描述
深海探险风格。背景使用极深的黑蓝（让人联想到深海沟），荧光青为主色模仿深海生物发光，珊瑚橙作为对比色点缀（让人联想到深海热泉口的管虫）。卡片边缘带有轻微发光效果，模拟海底生物荧光。整体神秘、科技感、探索氛围。

### 适合的 JS 特效
- **光标类型**: `follower` — 小型发光点 (8px)，带有青色荧光羽化

- **翻转动画**: 卡片 hover 时"气泡"浮动上移
- **revealStyle**: `slide-up` (从下方升起的动画，如气泡上浮)
- **额外特效**: 背景缓慢漂浮的发光粒子（模拟深海微生物）

### 布局模式
`light-tech`

### 额外 CSS 要点
- 背景叠加两层圆形渐变模拟深海透光效果
- 卡片 hover 时气泡浮动效果（向上轻浮 + 发光加深）
- 荧光青作为主要 hover 高亮
- 使用 `mix-blend-mode` 创造发光文字效果

## 主题 8：抹茶极简 (Matcha Zen)

### 图标
🍵

```
--accent:       #6b8e23  (抹茶绿)
--accent-soft:  rgba(107,142,35,0.06)
--bg:           #f7f6f1  (米白/和纸)
--surface:      #ffffff
--text:         #2d2a24  (墨色)
--muted:        #8f8a7e  (鼠色)
--border:       rgba(107,142,35,0.08)
```

### 设计风格描述
受日本侘寂美学启发的极简抹茶风格。米白和纸色背景，抹茶绿作为唯一点缀色。大量留白，极细的边框。排版使用宽松的字间距和行高，营造呼吸感。页面布局规整、干净、有禅意。每个元素都经过精心选择，不多不少。

### 适合的 JS 特效
- **光标类型**: `none` (无自定义光标，保持系统原生的简洁)
- **视差**: 关闭
- **翻转动画**: 无翻转，只有极淡的透明度变化

- **额外特效**: 无复杂特效，突出内容的安静力量

`classic` (经典布局配合留白设计更合适)

### 额外 CSS 要点
- 字体使用 `Noto Sans SC` / `Noto Serif SC` 等中日文字体
- 最大内容宽度可缩小到 800px 增强阅读感
- 卡片不加阴影，只有极细边框 (0.5px)
- hover 效果只有背景色微变
- 大量使用 `padding` 和字间距创造"空气感"

## CSS 变量映射清单

所有新主题需要实现以下 CSS 自定义属性（参考现有 `theme.css`）：

```css
--theme-bg            /* 页面背景色 */
--theme-surface       /* 卡片/模块背景色（半透明版本） */
--theme-surface-strong /* 卡片/模块背景色（实色版本） */
--theme-text          /* 主要文字颜色 */
--theme-muted         /* 次要文字颜色 */
--theme-border        /* 边框颜色 */
--theme-accent        /* 强调色/主色 */
--theme-accent-soft   /* 强调色半透明版本 */
--theme-card          /* 卡片背景色 */
--theme-header-bg     /* 导航栏背景色 */
--theme-header-border /* 导航栏边框色 */
--theme-btn-text      /* 按钮文字颜色 */
--theme-code-bg       /* 代码块背景色 */
--theme-shadow-border /* 边框阴影（用于卡片轮廓） */
--theme-shadow-elevation /* 抬升阴影 */
--theme-card-radius   /* 卡片圆角 */
--theme-card-border-hover /* 卡片 hover 边框色 */
/* 光标效果（follower 模式需要） */
--theme-cursor-border /* 光标边框 */
--theme-cursor-bg     /* 光标背景 */
--theme-cursor-glow   /* 光标发光 */
```

## 新增 revealStyle

当前系统支持 `slide-up`、`fade`、`slide-left`。
经过设计的8个主题中，**暗红哥特**主题使用了 `zoom`（缩放淡入），需要在 ThemeEffects.astro 和 theme.css 中新增此类型：

```css
/* zoom (crimson noir style) */
[data-reveal-style="zoom"] [data-reveal] {
  opacity: 0;
  transform: scale(0.92);
  transition: opacity 0.6s ease var(--reveal-delay, 0ms),
              transform 0.6s cubic-bezier(0.2, 0.7, 0.1, 1) var(--reveal-delay, 0ms);
}
[data-reveal-style="zoom"] [data-reveal].visible {
  opacity: 1;
  transform: scale(1);
}
```

同时需要在 `CursorStyle` 类型中确认是否已有 `zoom` — 当前 types 中只有 `slide-up | fade | zoom | slide-left`，`zoom` 已存在。

## 实现优先级建议

**第一梯队（视觉冲击力最强、优先实现）:**
1. 赛博霓虹 — 辨识度最高，与现有任何主题都不同
2. 毛玻璃 — 现代感强，与暗色/浅色都区分明显
3. 暗红哥特 — 强烈的个性化表达

**第二梯队（风格独特、补充多样性）:**
4. 极寒科技 — 展示工程/技术背景
5. 深海深渊 — 神秘氛围
6. 抹茶极简 — 极致的克制美学

**第三梯队（氛围感主题）:**
7. 暖色复古 — 温暖故事感
8. 森之 Canopy — 自然沉浸
